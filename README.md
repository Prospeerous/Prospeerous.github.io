## Overview

Modern, responsive portfolio website built with Bootstrap 5, featuring a dynamic theme switcher and smooth animations. Showcases expertise in healthcare AI, fintech solutions, cloud infrastructure, and full-stack development.

## Key Features

- **Responsive Design** - Optimized for all devices
- **Dark/Light Theme** - Seamless theme switching with persistent preference
- **Interactive Animations** - Smooth scroll animations using AOS library
- **Project Showcase** - Featured projects with live demos and GitHub links
- **Contact Form** - Integrated contact functionality

## Tech Stack

- HTML5 & CSS3
- Bootstrap 5
- JavaScript (Vanilla)
- Bootstrap Icons
- AOS (Animate On Scroll)

## Quick Start

### View Live

Visit the live portfolio at: **[https://prospeerous.github.io/](https://prospeerous.github.io/)**

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Prospeerous/Prospeerous.github.io.git

# Navigate to directory
cd Prospeerous.github.io

# Open with any local server or simply open index.html in your browser
```
